# create-pocket-app

Create a new Pocket app with one command.

## Usage

```bash
npx create-pocket-app my-app
npx create-pocket-app my-app --template react-todo
npx create-pocket-app my-app --framework vue
```

## Features

- Scaffold a new Pocket project instantly
- Multiple starter templates (React Basic, React Todo, and more)
- Framework selection (React, Vue, Expo)
- Automatic dependency installation
- TypeScript and Vite preconfigured

## Templates

| Template | Description |
|---|---|
| `react-basic` | Empty React starter with Pocket |
| `react-todo` | Complete todo app with React and Pocket |

## Options

- `--template <name>` — Use a specific project template
- `--framework <name>` — Choose a framework (react, vue, expo)
