# Pocket Playground

An interactive playground for exploring the Pocket local-first database. Create, update, delete, and query documents in real time with live reactive updates.

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/pocket-db/pocket/tree/main/examples/playground)

## Features

- **Todo List** — Full CRUD with insert, update, and delete operations
- **Live Queries** — Reactive subscriptions that update the UI automatically
- **Query Explorer** — Type filter JSON and see matching results in real time
- **Status Bar** — Live document count and completion stats

## Quick Start

```bash
npm install
npm run dev
```
